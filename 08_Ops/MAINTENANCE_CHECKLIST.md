# Maintenance Checklist

**Purpose:** Regular maintenance tasks to keep the workspace clean, current, and functional. Ensures no leads fall through cracks and all files stay actionable.

**How to Use:**
- Review and complete daily checklist first thing in work session
- Run weekly review every Friday or Monday
- Complete monthly review once per month (recommend first or last Friday)
- Check off items as completed, update last run date in header
- Create follow-up actions as needed from checklist findings

---

## Daily Checklist (5 min, start of day)

- [ ] Review **ACCOUNTS_INDEX.md** - Any stale accounts to re-engage? (>7 days no touch)
- [ ] Check **Next Steps** section of active opportunities - Are we on track with promised follow-ups?
- [ ] Scan person files with `[temperature:warm]` or `[temperature:hot]` - Any waiting >3 days for response?
- [ ] Review P1 items in opportunity/person files - Any past due?
- [ ] Update status of any files from interactions yesterday (new calls, emails, research)

## Weekly Checklist (30 min, Friday afternoon or Monday morning)

- [ ] **Pipeline review:** Open all opportunities marked `[stage:demo]` or later - what moved, what stalled?
- [ ] **Stale account sweep:** Find all accounts/people with `[status:stale]` or last touch >14 days - flag for re-engagement or archive
- [ ] **Index update:** Update ACCOUNTS_INDEX.md with new companies/people added this week, reflect current pipeline status
- [ ] **Win/loss analysis:** Review closed opportunities - what patterns emerge in wins vs. losses? Update messaging hypotheses?
- [ ] **Tagging audit:** Spot-check 5-10 files - are tags accurate and up-to-date? Stale tags removed?
- [ ] **Template consistency:** Did you use templates this week? Any needed updates to templates based on what you learned?
- [ ] **Opportunity risk assessment:** Review all active opportunities, update risk factors, assess probability
- [ ] **Warm intro pipeline:** Any mutual connections you haven't leveraged? Update relationship notes

## Monthly Checklist (60 min, last Friday of month)

- [ ] **Archive sweep:** Move all stale files (>30 days, no updates) to 99_Archive/[YYYY-MM]/
  - [ ] Update ARCHIVE_INDEX.md with reason for archiving
  - [ ] Remove from main ACCOUNTS_INDEX.md
- [ ] **Pipeline health score:** Count and assess:
  - [ ] How many accounts actively engaged?
  - [ ] How many in discovery vs. demo vs. proposal?
  - [ ] Average time to progression - trends improving or getting longer?
- [ ] **Process improvements:** Review your workflow - what manual processes could be templated? What new templates are needed?
- [ ] **Company research gaps:** Are there industries or company sizes you haven't researched? Flag for next month
- [ ] **Messaging effectiveness:** Which messaging angles are resonating? Update best performers in company files

## Common Findings & Actions

When you find issues during maintenance:

**Stale opportunity (>21 days in same stage):**
- [ ] Update `[status:waiting]` tag
- [ ] Note in activity log what's blocking progression
- [ ] Set explicit follow-up date with champion
- [ ] Escalate to manager if P1

**Warm lead with no recent touch (>7 days):**
- [ ] Add to daily outreach: quick check-in email or call
- [ ] Update communication history with outreach date
- [ ] Set calendar reminder for 3 days if no response

**Account moved to archive:**
- [ ] Move file to 99_Archive/[YYYY-MM]/filename.md
- [ ] Add entry to ARCHIVE_INDEX.md with reason
- [ ] Remove from ACCOUNTS_INDEX.md

**Outdated template fields:**
- [ ] Update template file with new field
- [ ] Make note in template changelog (optional subsection)
- [ ] Apply to next 2-3 new files created
