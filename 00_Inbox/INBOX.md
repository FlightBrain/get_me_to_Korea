# Inbox

> **Purpose:** Capture everything fast. Process later.
> **Last Updated:** 2026-04-01
> **Rule:** Nothing stays here longer than 1 week. Triage daily.

## How to Use

Drop anything here that doesn't have an obvious home yet:
- Raw ideas
- Pasted emails or messages
- LinkedIn profile notes
- Reddit thread links
- X thread summaries
- Meeting notes (before processing)
- Screenshots
- Random thoughts
- Links to read later

### Subfolders

| Folder | What Goes There |
|---|---|
| `raw/` | Unprocessed text, notes, brain dumps |
| `screenshots/` | Screen captures, images |
| `links/` | URLs to process later |
| `snippets/` | Copy-pasted text from anywhere |

### Triage Rules

When processing inbox items, route them to:

| If it's... | Send it to... |
|---|---|
| A preference or lesson | `01_Memory/` |
| A task or action item | `02_Tasks/TODAY.md` or `BACKLOG.md` |
| An outreach draft or idea | `03_Outreach/` |
| Research or signal | `04_Research/` |
| A good phrase or writing idea | `05_Writing/` |
| A repeatable process | `06_Playbooks/` |
| About a person or company | `07_Accounts/` |
| Stale or done | `99_Archive/` |
