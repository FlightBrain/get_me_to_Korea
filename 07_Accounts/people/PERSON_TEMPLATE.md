# Person Note Template

**Purpose:** Contact intelligence and relationship tracker for individual stakeholders. Maintains communication history and personality context.

**How to Use:**
- Create a new file for each person using the naming convention: `firstname_lastname.md`
- Update communication history after every interaction
- Note personality/communication style for personalization
- Reference best messaging angle when planning outreach
- Use for building champions and identifying decision makers
