# Session Log Template

**Purpose:** Capture work done in a session, outcomes, and learnings for continuity and pattern recognition. Create one session log file per work session.

**How to Use:**
- Create new file each session: `YYYY-MM-DD_session_log.md`
- Complete during or immediately after session while memory is fresh
- Reference at end of session when updating AI memory
- Allows AI assistant to understand what was done and build on it next session
- Review weekly to identify patterns and insights
