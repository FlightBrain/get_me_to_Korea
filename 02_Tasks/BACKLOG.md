# BACKLOG

## Purpose
Unprioritized task pool. Items here are valid work but not yet committed to a specific week. Use this as your "idea inbox" and to capture tasks that emerge during the week but don't fit TODAY or THIS_WEEK.

## How to Use
- Add any task that comes up that isn't urgent
- **To promote to THIS_WEEK**: Move item to weekly file, update status
- **To promote to TODAY**: Item is truly blocking (rare), move and notify manager
- Review BACKLOG every Friday to prioritize for next week
- Keep items brief (~5-10 words) with optional context notes
- Items in High Priority should be reviewed weekly; Medium/Low can be quarterly

## Promotion Guidelines
**Move to THIS_WEEK if:**
- Blocks another committed item
- Manager flagged it as priority
- Dependency cleared (e.g., waiting on tool access)
- Opportunity window closing (time-bound)

**Move to TODAY if:**
- Production issue / deal at risk
- Manager escalation
- Critical path blocker (very rare)
