# THIS_WEEK

## Purpose
Weekly snapshot of goals, daily focus areas, key deliverables, and meetings to prepare for. Provides structure across Mon-Fri and ensures nothing falls through cracks.

## How to Use
- Set these on Monday morning or Friday EOD (for next week)
- Limit to 3 week goals max to stay focused
- Update daily focus each evening for next day
- Use for 1:1 prep with manager
- Move completed items to DONE_LOG
- Carry forward uncompleted work to next week or BACKLOG
