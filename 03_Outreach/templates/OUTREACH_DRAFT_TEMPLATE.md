# Outreach Draft Template

## Purpose
Structured template for drafting any outreach message (email, LinkedIn, X, comment). Forces you to think through positioning, hook, and CTA before writing the final copy. Faster iteration than free-form drafting.

## How to Use
1. Fill out all fields below (even if roughly)
2. Write the message draft
3. Review against MESSAGING_PRINCIPLES.md checklist
4. Revise until it passes all six principles
5. Send or save to appropriate channel index
6. After sending, log it and track response

### **Pre-Send Checklist**

Before you send, verify:
- [ ] Subject line is specific (not generic)
- [ ] First sentence references a specific signal
- [ ] Message is under 100 words (for cold email) or under 3 sentences (for LinkedIn)
- [ ] CTA is specific and clear
- [ ] I can answer "why them?" in one sentence
- [ ] Tone matches their communication style
- [ ] No corporate jargon (synergize, leverage, etc.)
- [ ] If follow-up, I acknowledge non-response directly (no guilt, just honesty)
- [ ] I'm not pitching before they've asked about the problem
- [ ] This would make sense to someone who doesn't work at Braintrust

## Tips for Filling This Out

- **Audience:** Spend 2 minutes researching before filling this section. Signal research is the hardest part; get it right and the rest flows.
- **Goal:** Be specific. "Start conversation" is vague. "Ask how they test agent reliability" is specific.
- **Personalization Hook:** This is make-or-break. If you can't fill this section with something real, don't reach out yet.
- **Message Draft:** Write rough, then refine against MESSAGING_PRINCIPLES.md checklist.
- **Tone Notes:** Reading your draft out loud helps you catch when the tone is off.
