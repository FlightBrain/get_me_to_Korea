# Follow-Up Email Template

## Post-Meeting Follow-Up

- Voice: mirror recipient formality, short paragraphs, open with specific meeting reference
- Data: Granola meeting notes, Gmail thread, Slack #c-{company}
- Format: reference what THEY said, include braintrust.dev/customers link, clear next step
- Length: 6-10 sentences for ongoing, 3-5 for quick follow-ups
- Always include external links only (braintrust.dev, Loom recordings). Never internal Notion links.
- OK to mention the $80M Series B
- Include https://www.braintrust.dev/customers and https://www.braintrust.dev/trace when relevant

## Relationship Check-in

- Voice: warm, specific, brief. Must reference something real and recent.
- If no specific angle found: DO NOT DRAFT. Flag for Kensington.
- 3-6 sentences. Specific hook + value add + soft CTA.
- Anti-patterns: "just checking in", "wanted to touch base", "hope all is well"

## Event Follow-Up

- Reference the specific event they attended
- Drop a case study that maps to what was discussed
- Soft CTA for a deeper conversation
- Include event recording link if available

## Self-Review

- [ ] Meeting reference is accurate?
- [ ] Next steps match what was actually discussed?
- [ ] No stale commitments included?
- [ ] Length appropriate?
- [ ] No em dashes?
- [ ] HTML signature appended?
