# Braintrust SDR Playbook Index

Master index of all playbooks and SOPs. Each playbook is self-contained but references others where relevant.

## MAINTENANCE SOPs
Core daily/weekly routines that keep your system organized.

| Playbook | Purpose | Time | When to Use |
|----------|---------|------|------------|
| [DAILY_STARTUP.md](maintenance/DAILY_STARTUP.md) | Morning routine to set context and priorities | 15 min | Every morning, first thing |
| [DAILY_SHUTDOWN.md](maintenance/DAILY_SHUTDOWN.md) | End-of-day cleanup and tomorrow setup | 10 min | Every evening, before logging off |
| [WEEKLY_REVIEW.md](maintenance/WEEKLY_REVIEW.md) | Clean up backlogs, assess performance, plan week | 45 min | Every Friday afternoon |

## Quick Decision Tree

**I'm starting my day** → Use DAILY_STARTUP.md

**I need to find targets with pain** → Use REDDIT_MINING.md or X_SIGNAL_WORKFLOW.md

**I've found a prospect I like** → Use LINKEDIN_RESEARCH.md

**I'm ready to reach out** → Use RESEARCH_TO_OUTREACH.md → EMAIL_WRITING.md → CHANNEL_PLAYBOOK_EMAIL.md

**Follow-up strategy?** → See CHANNEL_PLAYBOOK_EMAIL.md follow-up cadence

**End of day** → Use DAILY_SHUTDOWN.md

**Friday afternoon** → Use WEEKLY_REVIEW.md
